Binary Search Algorithm

Instruction

  • Entered the number you want to search which will be considerd as key through out the algorithm
  • Firstly you will get low and high pointer in red color.
  • This algorithm works upon the formula Low+high/2
  • By using this formula compute the mid.
  • Then compare mid with key.
  • You will follow this steps 2 to 5 until the mid matches with key
  • when key matches with key it will been shown in green color
